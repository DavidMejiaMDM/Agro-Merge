## 📌 Sobre el Proyecto

**Agro Merge** Es una más que una propuesta de mercado ya que nuestro enfoque es ayudar a los agricultores independientes con sus
productos, obteniendo beneficios para sus cultivos, como mercado de el, buscamos abrir puertas al sector rural con nuestra plataforma
dando la oportunidad de abastecerse de insumos para el trabajo agrícola, directamente con los distribuidores.

### ✨ Diferenciador Clave
Este Sistema, busca fortalecer el sector agro con el sector mercado, facilitando comercios:
* **Cultivos Orgánicos:** enfocado a cultivos certificados que cumplan estándares de sanidad, producto calidad.
* **Cliente:** Conecta directamente con el mercado agro y Empresas de insumos químicos.
* **Empresas:** Facilita los suministros para los cultivos del agricultor independiente sin necesidad de intermediarios.
